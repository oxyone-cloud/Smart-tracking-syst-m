# 📍 SSCI Smart Tracking System

**SSCI Smart Tracking** is a real-time tracking and logistics monitoring platform developed by **SSCI Solution of Cold**.  
It enables live GPS tracking, temperature supervision, and route history for vehicles, cold rooms, and logistics assets — all powered by **Firebase** and modern web technologies.

🌐 **Live Application:** [https://studio-8743797555-41ff2.web.app/smart-tracking.html](https://studio-8743797555-41ff2.web.app/smart-tracking.html)

---

## 🚀 Key Features
- 📡 **Real-time GPS tracking** of vehicles and cold logistics units.  
- 🌡️ **Temperature monitoring** integrated with cold chain sensors.  
- 🚧 **Geofencing alerts** when an asset exits or enters a defined area.  
- 📊 **Route history** and CSV data export.  
- 🧭 **Responsive dashboard** accessible from any browser.  
- 🔐 Secure cloud data management via **Firebase Firestore**.

---

## 🧰 Technologies Used
- **JavaScript / HTML / CSS** (frontend)  
- **Firebase (Hosting & Firestore)**  
- **Leaflet & OpenStreetMap** (map engine)  
- **GPS Tracking APIs**  
- **Flutter integration (optional mobile version)**
